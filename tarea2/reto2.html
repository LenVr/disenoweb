Herramientas de Software

Algunas de las herramientas más usadas por los desarrolladores de software

Github una herramienta que funciona con la tecnología Git que ofrece un mejor trabajo colaborativo entre desarrolladores

Mongo DB es una base de datos no relacional o NoSQL, muy querida entre muchos desarrolladores y programadores

React es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página
